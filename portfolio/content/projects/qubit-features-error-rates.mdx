## Summary

An empirical study assessing correlations between qubit-level features (T1, T2 coherence times, gate fidelity, readout error) and overall circuit error rates, building a predictive model for qubit quality.

## Technical Approach

Using IBM Quantum hardware calibration data, we trained regression models to predict per-circuit error rates from qubit feature vectors.

<CodeBlock
  language="python"
  filename="qubit_analysis.py"
  code={`
import pandas as pd
from sklearn.ensemble import RandomForestRegressor

# Calibration data from IBMQ backends
data = pd.read_csv("qubit_calibration.csv")
features = ["T1", "T2", "gate_fidelity", "readout_error"]
X = data[features]
y = data["circuit_error_rate"]

model = RandomForestRegressor(n_estimators=100)
model.fit(X, y)

# Feature importance
importances = pd.Series(model.feature_importances_, index=features)
  `}
/>

## Results

Gate fidelity was the strongest predictor of circuit error rates, followed by T2 coherence time. The model achieved an R² of 0.74 on held-out test data.
