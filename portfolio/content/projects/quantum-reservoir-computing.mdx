## Summary

This project explores the application of QuEra's neutral-atom quantum architecture and recent quantum reservoir computing (QRC) techniques to predict molecular activity. The model compares the performance of QRC using simulation and real quantum hardware against classical ML models on the same task.

## Problem & Constraints

Quantum reservoir computing offers a promising pathway to quantum advantage on near-term hardware by using quantum systems as fixed, randomly initialized dynamical systems — the "reservoir" — and training only a classical readout layer. The key constraint: neutral-atom architectures have different connectivity and gate fidelities compared to superconducting qubits, requiring the reservoir design to be adapted to the underlying hardware topology.

<Callout type="info">
Neutral-atom quantum computers (like QuEra's Aquila) use individually trapped atoms manipulated by laser arrays. They excel at native analog simulation but require different algorithmic strategies than gate-model machines.
</Callout>

## Technical Approach

The reservoir was constructed using a network of interacting neutral atoms, where the interaction Hamiltonian governs the dynamics. Input molecular features were encoded as initial atomic states, and the reservoir's evolution was measured to produce feature vectors for a classical linear classifier.

import numpy as np
from quera.analog import AnalogQuantumComputer

class QuantumReservoir:
    def __init__(self, n_qubits: int, interaction_strength: float):
        self.n_qubits = n_qubits
        self.qpu = AnalogQuantumComputer()
        self.interaction = interaction_strength
    
    def encode_input(self, features: np.ndarray) -> None:
        """Encode molecular features as initial atomic states."""
        # Map features to initial Rydberg state populations
        pulses = self._features_to_pulses(features)
        self.qpu.apply_pulse_sequence(pulses)
    
    def evolve(self, time: float) -> np.ndarray:
        """Evolve the reservoir and measure final states."""
        self.qpu.evolve(time)
        return self.qpu.measure_all()
    
    def transform(self, X: np.ndarray) -> np.ndarray:
        """Transform input features into reservoir states."""
        features = []
        for sample in X:
            self.encode_input(sample)
            state = self.evolve(time=1.0)
            features.append(state)
        return np.array(features)
  `}

## Key Implementation Details

The reservoir transformation creates a high-dimensional feature space that makes classification easier for the downstream linear model. The key insight: the quantum dynamics naturally maps similar inputs to nearby reservoir states, preserving the geometric structure of the data while expanding its dimensionality.

<Callout type="tip">
The choice of interaction time is critical — too short and the reservoir doesn't mix enough; too long and the dynamics become chaotic, losing discriminative information.
</Callout>

## Results

Benchmarking against classical methods (SVM, Random Forest, k-NN) on the same molecular activity dataset:

- **QRC (simulation)**: 89.2% accuracy
- **QRC (hardware)**: 84.7% accuracy  
- **SVM**: 87.1% accuracy
- **Random Forest**: 85.3% accuracy
