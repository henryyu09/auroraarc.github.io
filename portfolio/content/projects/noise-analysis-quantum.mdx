## Summary

This project systematically analyzed how different noise channels and noise levels affect the classification accuracy of quantum machine learning models, establishing noise thresholds for reliable quantum classification.

## Technical Approach

Variational quantum classifiers were trained on synthetic datasets under various IBM Qiskit noise models, including depolarizing, amplitude damping, and phase damping channels at multiple noise strengths.

from qiskit.providers.aer import QasmSimulator
from qiskit.providers.aer.noise import NoiseModel

def analyze_noise_sensitivity(model, noise_levels):
    results = []
    for level in noise_levels:
        noise_model = NoiseModel()
        noise_model.add_all_qubit_quantum_error(
